# Sales Performance Analysis

## Objective
Analyze sales data to identify high-performing regions, profitable categories, sales trends, and top products.

## Tools Used
- SQL (MySQL)
- Microsoft Excel
- Power BI

## Key Analysis
- Region-wise total sales
- Category-wise profit analysis
- Monthly sales trend
- Top products by revenue

## Workflow
1. Performed data analysis using SQL queries.
2. Validated insights using Excel Pivot Tables.
3. Built an interactive Power BI dashboard with KPIs and slicers.

## Outcome
Delivered actionable business insights through a clean and interactive dashboard.
